🧮 Mathematics

  • Important Derivative Formulas:
    • \[ \frac{d}{dx}(x^n) = nx^{n-1} \]
    • \[ \frac{d}{dx}(e^x) = e^x \]
    • \[ \frac{d}{dx}(a^x) = a^x \ln a \]
    • \[ \frac{d}{dx}(\ln x) = \frac{1}{x} \]
    • \[ \frac{d}{dx}(\sin x) = \cos x \]
    • \[ \frac{d}{dx}(\cos x) = -\sin x \]
    • \[ \frac{d}{dx}(\tan x) = \sec^2 x \]
    • \[ \frac{d}{dx}(\cot x) = -cosec^2 x \]
    • \[ \frac{d}{dx}(\sec x) = \sec x \tan x \]
    • \[ \frac{d}{dx}(cosec x) = -cosec x \cot x \]
  • Important Derivatives of Inverse Functions:
    • \[ \frac{d}{dx}(\sin^{-1}x) = \frac{1}{\sqrt{1-x^2}} \]
    • \[ \frac{d}{dx}(\cos^{-1}x) = -\frac{1}{\sqrt{1-x^2}} \]
    • \[ \frac{d}{dx}(\tan^{-1}x) = \frac{1}{1+x^2} \]
    • \[ \frac{d}{dx}(\cot^{-1}x) = -\frac{1}{1+x^2} \]
    • \[ \frac{d}{dx}(\sec^{-1}x) = \frac{1}{|x|\sqrt{x^2-1}} \]
    • \[ \frac{d}{dx}(cose c^{-1}x) = -\frac{1}{|x|\sqrt{x^2-1}} \]
  • Standard Trigonometric Formulas:
    • \[ \sin 2x = 2\sin x \cos x \]
    • \[ \cos 2x = \cos^2 x - \sin^2 x \]
    • \[ \tan 2x = \frac{2\tan x}{1 - \tan^2 x} \]
    • \[ \sin x = 2\sin\left( \frac{x}{2} \right)\cos\left( \frac{x}{2} \right) \]
    • \[ \cos x = \cos^2\left( \frac{x}{2} \right) - \sin^2\left( \frac{x}{2} \right) \]
    • \[ \sin^2 x + \cos^2 x = 1 \]
  • Standard Hyperbolic Formulas:
    • \[ \sinh 2x = 2\sinh x \cosh x \]
    • \[ \cosh 2x = \cosh^2 x + \sinh^2 x \]
    • \[ \sinh x = 2\sinh\left( \frac{x}{2} \right)\cosh\left( \frac{x}{2} \right) \]
    • \[ \cosh x = \cosh^2\left( \frac{x}{2} \right) + \sinh^2\left( \frac{x}{2} \right) \]
    • \[ \cosh^2 x - \sinh^2 x = 1 \]
  • Hyperbolic Functions:
    • \[ \int \sinh x \, dx = \cosh x + C \]
    • \[ \int \cosh x \, dx = \sinh x + C \]
    • \[ \int \tanh x \, dx = \ln|\cosh x| + C \]
    • \[ \int \text{sech}^2 x \, dx = \tanh x + C \]
    • \[ \int \text{cosech}^2 x \, dx = -\coth x + C \]
    • \[ \int \cot x \, dx = \ln|\sin x| + C \]
  • Important Derivatives of Inverse Hyperbolic Functions:
    • \[ \frac{d}{dx}(\sinh^{-1} x) = \frac{1}{\sqrt{1+x^2}} \]
    • \[ \frac{d}{dx}(\cosh^{-1} x) = \frac{1}{\sqrt{x^2-1}} \quad (x > 1) \]
    • \[ \frac{d}{dx}(\tanh^{-1} x) = \frac{1}{1-x^2} \quad (|x| < 1) \]
    • \[ \frac{d}{dx}(\coth^{-1} x) = \frac{1}{1-x^2} \quad (|x| > 1) \]
    • \[ \frac{d}{dx}(\text{sech}^{-1} x) = \frac{-1}{x\sqrt{1-x^2}} \quad (0 < x \leq 1) \]
    • \[ \frac{d}{dx}(\text{cosech}^{-1} x) = \frac{-1}{|x|\sqrt{1+x^2}} \quad (x \neq 0) \]
  • Standard Integration Formulas:
    • \[ \int \frac{dx}{a^2 + x^2} = \frac{1}{a} \tan^{-1}\left( \frac{x}{a} \right) + C \]
    • \[ \int \frac{dx}{a^2 - x^2} = \frac{1}{2a} \ln\left| \frac{a+x}{a-x} \right| + C \]
    • \[ \int \frac{dx}{x^2 - a^2} = \frac{1}{2a} \ln\left| \frac{x-a}{x+a} \right| + C \]
  • Standard Integration Formulas (With Square Root):
    • \[ \int \frac{dx}{\sqrt{a^2 + x^2}} = \ln\left| x + \sqrt{a^2 + x^2} \right| + C \]
    • \[ \int \frac{dx}{\sqrt{a^2 - x^2}} = \sin^{-1}\left( \frac{x}{a} \right) + C \]
    • \[ \int \frac{dx}{\sqrt{x^2 - a^2}} = \ln\left| x + \sqrt{x^2 - a^2} \right| + C \]
  • Important Integration Formulas:
    • \[ \int x^n \, dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1) \]
    • \[ \int e^x \, dx = e^x + C \]
    • \[ \int a^x \, dx = \frac{a^x}{\ln a} + C \quad (a>0, a\neq1) \]
    • \[ \int \frac{1}{x} \, dx = \ln|x| + C \]
    • \[ \int \sin x \, dx = -\cos x + C \]
    • \[ \int \cos x \, dx = \sin x + C \]
    • \[ \int \sec^2 x \, dx = \tan x + C \]
    • \[ \int \text{cosec}^2 x \, dx = -\cot x + C \]
    • \[ \int \sec x \tan x \, dx = \sec x + C \]
    • \[ \int \text{cosec} x \cot x \, dx = -\text{cosec} x + C \]
  • Full Integration by Parts Formula:
    \[ \int uv \, dx = u \int v \, dx - \int \left( u' \int v \, dx \right) dx \]
